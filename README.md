# Maplyo
The E-guide for every Airbnb

## Install
```bash
npm install
npm run dev
```

## Routes
- `/` marketing landing
- `/auth/login` and `/auth/signup` placeholders
- `/app/guides` dashboard demo
- `/app/guides/demo/builder` builder MVP
- `/g/demo` public guide

## Notes
This is a clean greenfield starter. We'll add Supabase DB + RLS, Stripe, themes, DnD builder, and many more blocks next.
